# Revnue-Data-and-Building-a-Dashboard
Python-based financial data analysis project analyzing stock and revenue data for Tesla, Amazon, AMD, and GameStop. Implemented data extraction (APIs &amp; web scraping), data cleaning, and interactive dashboard visualization using Plotly

I analyzed historical stock and revenue data for:
Tesla
Amazon
Advanced Micro Devices
GameStop

What I Did:
Extracted stock data using APIs
Collected revenue data using web scraping (BeautifulSoup)
Cleaned and prepared datasets using Pandas
Analyzed stock price and revenue trends
Built an interactive dashboard using Plotly

Tools Used:
Python
Pandas
NumPy
Requests
BeautifulSoup
Plotly
Jupyter Notebook

Key Outcome:
This project demonstrates my ability to perform end-to-end data analysis, from data extraction to visualization, using Python and data science libraries.
